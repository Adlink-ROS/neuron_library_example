# Neuron Library ROS 2 Example

This is a neuron library ROS 2 example for ADLINK's products.

## Environment Requirement
* ROS version: 
  - Foxy
* Ubuntu version:
  - 18.04
  - 20.04
* Hardware support list:
  - ROSCube-I
  - ROSCube-X
